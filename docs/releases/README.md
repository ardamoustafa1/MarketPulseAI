# Release Notes Discipline

Her PR, `docs/releases/` altina bir not eklemelidir.

## Dosya formati

- `YYYY-MM-DD-kisa-baslik.md`
- Icerik:
  - Problem
  - Cozum
  - Trade-off / risk
  - Test ve rollout notu

## Minimal template

```md
# <title>

## Problem
- ...

## Change
- ...

## Risk / Trade-off
- ...

## Validation
- ...
```
